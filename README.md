# hithesis-template

已初步配置完成的 [hithesis](https://github.com/dustincys/hithesis) 毕业论文写作模板。

推荐（在对应目录下）使用 `latexmk` 构建。

* `opening` 开题报告
* `midterm` 中期报告
* `thesis` 毕业论文
